# Farm Input Authenticity & Crop Monitoring Platform

A comprehensive MERN stack platform designed to help farmers verify the authenticity of agricultural inputs (seeds, fertilizers, pesticides) and monitor crop health using AI.

## 🚀 Features

- **KrishiBot AI**: A cosmic-themed agricultural assistant (powered by Moonshot AI/Gemini) to answer farming queries.
- **Counterfeit Reporting**: Formally report suspicious or fake agricultural products to authorities.
- **Product Verification**: Scan and verify agricultural inputs for authenticity.
- **Crop Analysis**: Image-based AI analysis for pest and disease management.
- **Multilingual Support**: Available in English, Hindi, and Telugu.
- **Cosmic UI**: Premium, interactive interface with gravity-defying aesthetics and particle effects.

## 🛠️ Technology Stack

- **Frontend**: React, Vite, Tailwind CSS, Lucide Icons, Framer Motion.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB.
- **AI Integration**: Hugging Face (Moonshot AI), Google Gemini AI.

## 📦 Setup Instructions

### Backend
1. Navigate to the `backend` folder.
2. Install dependencies: `npm install`.
3. Create a `.env` file based on the provided configuration.
4. Run the development server: `npm run dev`.

### Frontend
1. Navigate to the `frontend` folder.
2. Install dependencies: `npm install`.
3. Run the development server: `npm run dev`.
